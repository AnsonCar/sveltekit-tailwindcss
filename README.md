# sveltekit-tailwind
## Introduce
This is a sample project built using sveltekit and tailwindcss (& daisyui).

## Strat up
```bash
npm i 
```

## Developing
```bash
npm run dev
# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building
```bash
npm run build
node ./build/index.js
```
